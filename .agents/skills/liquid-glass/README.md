# Liquid Glass Reference
iOS 26 Liquid Glass: Ultimate Swift/SwiftUI Reference. 
Source: https://github.com/conorluddy/LiquidGlassReference

## Summary
This skill provides the design language and technical principles for "Liquid Glass" aesthetics. While the original reference is for SwiftUI, this skill interprets the principles for use in Next.js/React projects, focusing on backdrop-filters, refraction shaders, and fluid spring animations.

## Usage
Simply invoke "Liquid Glass" or "make it glassy" to have the agent apply these specific iOS 26 style principles to your components.
